# Quick Start - Modual

Start binnen 5 minuten met Modual!

## Stap 1: Environment Setup (2 minuten)

Maak een `.env.local` bestand in de project root:

```env
JWT_SECRET=mijn-super-geheime-sleutel-123456
OPENAI_API_KEY=sk-jouw-openai-api-key (optioneel)
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## Stap 2: Installeer & Start (2 minuten)

```bash
npm install
npm run dev
```

## Stap 3: Open de App (1 minuut)

Ga naar: **http://localhost:3000**

## Je Eigen Logo Gebruiken

De app gebruikt momenteel een tijdelijk SVG logo. Om je eigen logo te gebruiken:

1. Kopieer je logo naar `public/logo.png` (of logo.svg)
2. Herstart de dev server

Het logo zal automatisch verschijnen op alle pagina's.

## Eerste Test

1. **Registreer**: Maak een test account aan
